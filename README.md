# Atom

Atom is a .NET-based Discord utility or bot project.

## Features

- **Discord Service**: Integrated Discord functionality using `Discord.Net`.
- **UI Helpers**: Custom utilities for UI management.
- **Models**: Structured data models for Discord interactions.

## Project Structure

- `Atom/`: Main project folder.
  - `Models/`: Data structures.
  - `Services/`: Business logic and Discord integration.
  - `Utils/`: Helper methods and UI utilities.
- `Atom.slnx`: Visual Studio Solution file.

## Technologies

- .NET 10.0
- Discord.Net
- Newtonsoft.Json

## Getting Started

1. Clone the repository.
2. Open `Atom.slnx` in Visual Studio 2022 or later.
3. Restore NuGet packages.
4. Build and Run.
